Taxi Demand Forecasting

Raw Data Information

About the Raw Data

The raw data consists of NYC Yellow Taxi trip records from January 2015. This data is the foundation of our taxi demand forecasting project.

Data Source

The original data can be downloaded from the NYC Taxi & Limousine Commission (TLC) website:

Download from TLC
Data Structure

The raw data file (yellow_tripdata_2015-01.csv) contains the following columns:

Column Name Description Data Type
pickup_datetime Date and time when the passenger was picked up datetime
dropoff_datetime Date and time when the passenger was dropped off datetime
pickup_longitude Longitude coordinate of pickup location float
pickup_latitude Latitude coordinate of pickup location float
dropoff_longitude Longitude coordinate of dropoff location float
dropoff_latitude Latitude coordinate of dropoff location float
passenger_count Number of passengers in the vehicle integer
trip_distance Distance of the trip in miles float
Data Processing

This raw data is processed in the following steps:

  1. Data cleaning and validation
  2. Conversion to grid-based format
  3. Feature engineering
  4. Preparation for model training
Note: The raw data file is large (1.8GB) and is not included in the GitHub repository. Please download it from the TLC website.
