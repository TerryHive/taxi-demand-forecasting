Taxi Demand Forecasting

Interim Data Information

About the Interim Data

The interim data consists of processed and cleaned datasets that have been transformed from the raw data. These datasets are ready for feature engineering and model preparation.

Available Datasets
Dataset Name Description Format
cleaned_trips.csv Cleaned taxi trip records with validated coordinates and timestamps CSV
grid_demand.csv Hourly demand counts for each grid cell CSV
weather_processed.csv Processed weather data aligned with trip timestamps (Planned for future analysis) CSV
holidays_processed.csv Processed holiday and event information (Planned for future analysis) CSV
Processing Steps
  1. Data Cleaning
    • Removal of invalid coordinates
    • Handling of missing values
    • Outlier detection and treatment
  2. Data Transformation
    • Conversion to grid-based format
    • Time-based aggregation
    • Feature normalization
  3. Data Integration
    • Merging with weather data
    • Adding holiday information
    • Creating temporal features
Note: These processed datasets are generated from the raw data and are used as input for the model training process.
